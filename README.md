### Запуск проекта
1. Создать `.development.env` и/или `.production.env` из `.env.example`
2. Прописать конфиги postgresql
3. Установить зависимости `yarn`
4. Запустить проект `yarn start:dev`
5. Ткнуть api, создающий все вариации blood
